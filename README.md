# 💰 Wallet API

Простое REST API для управления кошельками и финансовыми операциями.

Учебный проект — бэкенд-сервис с поддержкой нескольких кошельков, пополнения, списания и просмотра баланса. Написан в рамках подготовки к позиции Python Backend Developer.

## Возможности

- 📂 Создание именованных кошельков с начальным балансом
- 📈 Добавление доходов с описанием
- 📉 Списание расходов с проверкой остатка
- 🔍 Просмотр баланса — отдельного кошелька или общего
- ✅ Валидация входных данных через Pydantic v2

## Эндпоинты

| Метод | Путь | Описание |
|-------|------|----------|
| `GET` | `/balance` | Общий баланс или баланс конкретного кошелька |
| `POST` | `/wallet` | Создать новый кошелёк |
| `POST` | `/operations/income` | Пополнить кошелёк |
| `POST` | `/operation/expense` | Списать с кошелька |

## Быстрый старт

```bash
pip install fastapi uvicorn
uvicorn main:app --reload
```

Документация доступна по адресу: http://localhost:8000/docs

## Стек

- **FastAPI** — веб-фреймворк
- **Pydantic v2** — валидация данных
- **Uvicorn** — ASGI-сервер
- **Python 3.10+**
